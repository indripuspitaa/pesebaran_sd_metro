# pesebaran_sd_metro
Persebaran Sekolah Dasar (SD) merujuk pada distribusi spasial atau tata letak lembaga pendidikan tingkat dasar di suatu wilayah. Sebagai pilar utama wajib belajar, keberadaan SD dirancang untuk menjangkau seluruh lapisan masyarakat agar setiap anak usia 7–12 tahun mendapatkan hak pendidikan dasarnya.
